一、我的世界基岩版下载教程
①Android、HarmonyOS
1.下载https://klpbbs.com/xz/new/中的最新正式版apk安装包
2.安装apk安装包
②iOS、iPadOS
1.打开App Store
2.点击右上角头像
3.划到最底部退出登录（若不知道Apple ID密码请勿退出！）
4.登录下载账号（账号请自行购买，拼多多、淘宝等都有，超过一块你就亏死了）
5.搜索Minecraft并下载
6.下载完后重复步骤2-3
7.登录回自己的账号（不记得账号可在设置中的Apple ID查看）
特别提醒：不要退出设置里的Apple ID！若登录时出现Apple ID安全请选择 其他-不升级 ！
③Windows10、Windows11
1.下载https://share.weiyun.com/LhDTdOyg中的“缪斯星Minecraft Start Up”应用程序
2.打开Microsoft Store
3.搜索Minecraft for Windows
4.点击获取试玩版（若未登录微软账号则无法下载）
5.下载完试玩版后打开“缪斯星Minecraft Start Up.exe”
6.点击启动游戏即可游玩
特别提醒：杀毒软件报毒为误报，请将同意启动器所有操作！可将启动器加入杀毒软件信任区中。
特别帮助：使用此启动器可能导致重启电脑后不显示桌面，若出现此状况不必担心，请查看“缪斯星我的世界基岩版故障修复帮助”文档。

二、我的世界基岩版注册教程
⑴电子邮箱注册（本教程提供注册网易163邮箱，若已有网易‹我的世界网易版的网易邮箱也可以哦›或其他邮箱请跳到⑵步骤）
1.打开网址https://mail.163.com/register/
2.按照网站要求注册网易163邮箱
3.登录注册的邮箱
⑵微软账号注册
1.打开网址https://signup.live.com/
2.按照网站要求注册微软账号（请不要使用手机号注册，手机号注册会有更多注册步骤，请使用邮箱注册）
（设置游戏昵称时最好不要使用随机昵称，这样会不方便辨别各个玩家）
⑶登录微软账号
1.打开我的世界基岩版
2.点击左下角登录按钮
3.登录注册的微软账号
